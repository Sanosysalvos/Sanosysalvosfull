# Frontend
Interfaz de usuario de la aplicación.

## Tecnologías
- Next.js / React (basado en `app/`, `package.json`, `next.config.ts`)
- TypeScript
- Tailwind CSS (basado en `postcss.config.mjs`)

## Estructura
- `app/`: Rutas y páginas de la aplicación (App Router).
- `components/`: Componentes de UI reutilizables.
- `context/`: Gestión de estado global.
- `lib/`: Funciones y utilidades compartidas.
- `services/`: Servicios de comunicación con la API (BFF).
- `public/`: Archivos estáticos.
