# BFF (Backend For Frontend)
Este módulo actúa como la capa de orquestación entre el frontend y los microservicios.

## Tecnologías
- Java / Spring Boot (basado en la presencia de `pom.xml` y `mvnw`)
- Maven

## Estructura
- `src/`: Código fuente de la aplicación.
- `pom.xml`: Configuración de dependencias de Maven.
- `dockerfile`: Configuración para la contenedorización.
